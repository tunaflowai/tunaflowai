# TunaFlowAI v0.4 Upgrade Notes

This release adds three product-level foundations:

```text
1. Persona-based initialization
2. Installable/acquired job skills
3. Localhost dashboard GUI
```

## Persona-based initialization

New files:

```text
src/personas/persona-manager.js
src/personas/bundled/*/PERSONA.md
```

New CLI:

```bash
node src/cli.js personas
node src/cli.js persona active
node src/cli.js persona set anna
```

New API:

```text
GET  /personas
GET  /personas/active
POST /personas/:name/activate
POST /personas/switch
```

## Acquired job skills

New skill commands:

```bash
node src/cli.js skills create <name>
node src/cli.js skills acquire <path-or-loaded-skill-name>
node src/cli.js skills acquired
```

New API:

```text
GET  /skills/acquired
GET  /skill-jobs
POST /skills/acquire
POST /skills/install
POST /skills/reload
```

Acquired skills are copied to:

```text
.tunaflow/skills/acquired
```

## Dashboard

New file:

```text
src/dashboard/dashboard.js
```

New routes:

```text
GET /
GET /dashboard
GET /overview
```

## Compatibility

Existing v0.3 model provider, skill selector, channel adapter, approval, and audit APIs remain compatible.

Expected checks:
